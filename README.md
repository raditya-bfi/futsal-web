# Futsal Gembira

A brief description of what this project does and who it's for

## Run Locally

Clone the project

```bash
  git clone https://gitlab.com/transnovasi/k3i/main-web.git
```

Go to the project directory

```bash
  cd main-web
```

Install dependencies

```bash
  yarn install
```

Start the DEV server

```bash
  yarn dev
```

## **Options ✍️**

Check lint

```
yarn lint
```

Fix lint

```
yarn lint:fix
```

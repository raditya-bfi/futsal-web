# Futsal Gembira

## Run Locally

Clone the project

```bash
  git clone https://github.com/raditya-bfi/futsal-web.git
```

Go to the project directory

```bash
  cd futsal-web
```

Install dependencies

```bash
  yarn install
```

Start the Local server with .env

```bash
  yarn start
```

Start the DEV server .env.dev

```bash
  yarn dev
```

## **Options ✍️**

Check lint

```
yarn lint
```

Fix lint

```
yarn lint:fix
```

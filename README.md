# Futsal Gembira

## Run Locally

Clone the project

```bash
  git clone https://github.com/raditya-bfi/futsal-web.git
```

Go to the project directory

```bash
  cd main-web
```

Install dependencies

```bash
  yarn install
```

Start the DEV server

```bash
  yarn dev
```

## **Options ✍️**

Check lint

```
yarn lint
```

Fix lint

```
yarn lint:fix
```
